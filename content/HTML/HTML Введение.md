# HTML

**HTML** (HyperText Markup Language) — это язык разметки, который используется для создания и структурирования содержимого веб-страниц.  

### Основные моменты:
1. **Не является языком программирования** — это язык разметки, который определяет структуру контента (текст, изображения, ссылки, таблицы и т.д.).  
2. **Создаёт «скелет» страницы** — с помощью HTML вы размечаете, где будут заголовки, параграфы, списки, формы и другие элементы.  
3. **Работает вместе с CSS и JavaScript**:  
   - **CSS** отвечает за внешний вид (цвета, шрифты, расположение).  
   - **JavaScript** добавляет интерактивность (анимации, реакции на действия пользователя).  

### Для чего нужен HTML?
- **Структурирование веб-контента** — браузер читает HTML-код и отображает текст, изображения, видео в нужном порядке.  
- **Создание гиперссылок** — позволяет связывать страницы между собой (это основа «всемирной паутины»).  
- **Оптимизация для SEO** — правильная HTML-разметка помогает поисковым системам понимать содержание страницы.  
- **Доступность** — семантические теги (например, `<header>`, `<nav>`, `<article>`) помогают скринридерам корректно работать с сайтом для людей с ограниченными возможностями.  

### Простой пример HTML-кода:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это абзац текста на веб-странице.</p>
    <a href="https://example.com">Это ссылка</a>
</body>
</html>
```

### Ключевые термины:
- **Теги** — элементы разметки в угловых скобках (например, `<p>`, `<div>`).  
- **Атрибуты** — дополнительные свойства тегов (например, `href` у ссылки или `src` у изображения).  
- **Семантическая вёрстка** — использование тегов по смыслу (например, `<article>` для статьи, `<nav>` для навигации).  

**Итог**: HTML — фундаментальная технология в веб-разработке, без которой невозможно создать ни один сайт. Он задаёт структуру, которую затем стилизуют и оживляют с помощью CSS и JavaScript.
