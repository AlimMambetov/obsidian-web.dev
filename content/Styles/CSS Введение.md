# CSS 

**CSS-стили** — это правила оформления, которые определяют, как будут выглядеть HTML-элементы на веб-странице. CSS (Cascading Style Sheets, каскадные таблицы стилей) отделяет **содержание** (HTML) от **визуального представления**.

---

### **Что это такое?**
CSS состоит из **набора правил**, каждое из которых включает:
1. **Селектор** — указывает, к какому элементу применить стиль.
2. **Свойства и значения** — определяют конкретные параметры оформления.

**Пример:**
```css
h1 {
  color: blue;
  font-size: 24px;
  margin: 10px;
}
```
Здесь `h1` — селектор, а в фигурных скобках — свойства, меняющие цвет, размер шрифта и отступы.

---

### **Зачем они нужны?**
1. **Отделение структуры от оформления**  
   HTML отвечает за разметку (заголовки, параграфы, списки), а CSS — за внешний вид. Это упрощает редактирование и поддержку кода.

2. **Единообразие дизайна на всех страницах**  
   Один CSS-файл может применяться ко многим страницам сайта. Изменив стиль в одном месте, вы автоматически обновите весь сайт.

3. **Гибкое управление отображением**  
   CSS позволяет адаптировать дизайн под разные устройства (компьютеры, планшеты, телефоны) с помощью **медиа-запросов**.

4. **Эффекты и анимации**  
   Можно создавать интерактивные элементы: плавные переходы, анимацию, изменения при наведении курсора.

5. **Ускорение загрузки страниц**  
   Стили кэшируются браузером, поэтому при переходе между страницами сайта CSS-файл загружается один раз.

6. **Доступность и SEO**  
   Правильное использование CSS улучшает доступность контента для людей с ограниченными возможностями и помогает поисковым системам лучше анализировать сайт.

---

### **Пример разделения HTML и CSS:**
**HTML (структура):**
```html
<h1>Привет, мир!</h1>
<p>Это пример текста.</p>
```

**CSS (оформление):**
```css
h1 {
  color: darkgreen;
  text-align: center;
}
p {
  font-family: Arial, sans-serif;
  line-height: 1.5;
}
```

---

### **Типы подключения CSS:**
1. **Внешний файл** (рекомендуется) — стили в отдельном файле `.css`, подключаемом через `<link>` в HTML.
2. **Внутри HTML** — стили в теге `<style>` в разделе `<head>`.
3. **Инлайн-стили** — прямо в атрибуте `style` элемента (используется редко, так как усложняет поддержку).

---

### **Итог:**
CSS — **обязательная технология** веб-разработки, которая превращает «сырой» HTML в эстетичный, удобный и адаптивный интерфейс. Без CSS сайты выглядели бы как простые текстовые документы с гиперссылками.
