## **Теория:**

**Основные методы:**
- `querySelector()` - найти элемент
- `innerHTML` - содержимое элемента
- `createElement()` - создать элемент
- `appendChild()` - добавить элемент
- `addEventListener()` - обработать событие

 **Оператор ( ?. ) :**
- Позволяет безопасно обращаться к свойствам объектов
- Если значение слева равно `null` или `undefined`, возвращает `undefined` вместо ошибки

**Метод trim():**
- Удаляет пробелы в начале и конце строки
- `" текст ".trim()` → `"текст"`

**Массив** - список значений: `["значение1", "значение2"]`
- `push()` - добавить элемент
- `forEach()` - перебрать элементы

**Условие if:**
Проверяет условие и выполняет код, если оно истинно:

```js
if (условие) {
    // выполнить этот код
}

// ----- пример -----

if (age >= 18) console.log("Совершеннолетний");
if (!name) return; // Если name пустое, выйти из функции
```

## **Практика с DOM:**

```html
<body>
	<h1 id="title">Мой список дел</h1>
	<div id="todo-list"></div>  <!-- Здесь появится список -->
	<button id="btn">Добавить дело</button>  <!-- Кнопка для добавления -->
</body>
```

```css
body {
    font-family: Arial, sans-serif;  /* Шрифт для всей страницы */
    padding: 20px;  /* Отступы от краёв */
}

.item {
    padding: 10px;  /* Отступы внутри каждого дела */
    margin: 5px;    /* Отступы между делами */
    border: 1px solid #ccc;  /* Рамка вокруг дела */
    border-radius: 5px;  /* Закругленные углы */
}

/* Чётные дела будут серыми */
.item:nth-child(odd) {
    background-color: #f3f2f2;
}

button {
    padding: 4px 12px;  /* Размер кнопки */
    margin-top: 10px;   /* Отступ сверху */
}
```

```js
// Находим элементы на странице
const todoList = document.querySelector('#todo-list');
const button = document.querySelector('#btn');

// Начальный список дел
const todos = ["Позавтракать", "Пойти на работу", "Выучить JavaScript"];

// Показываем список при загрузке
renderTodos();

// При клике на кнопку - добавляем новое дело
button.addEventListener('click', addTask);

// Функция добавления дела
function addTask() {
    const newTodo = prompt('Введите новое дело:')?.trim();
    if (!newTodo) return;  // Если пусто - выходим
    
    todos.push(newTodo);  // Добавляем в массив
    renderTodos();  // Обновляем список
}

// Функция отрисовки списка
function renderTodos() {
    todoList.innerHTML = '';  // Очищаем старый список
    
    // Для каждого дела создаём элемент
    todos.forEach((el, i) => {
        const item = document.createElement('div');
        item.classList.add('item');
        item.textContent = `${i + 1}. ${el}`;  // "1. Позавтракать"
        todoList.appendChild(item);
    });
}
```
