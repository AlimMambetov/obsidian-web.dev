## **Теория:**

**Переменные:**
- `let` — для переменных, которые можно менять
- `const` — для констант (нельзя менять)
- `var` — устаревший способ (не используйте)

**Типы данных:**
- `string` — текст (например: `"Привет"`)
- `number` — числа (например: `5`, `3.14`)
- `boolean` — логический тип (`true`/`false`)
- `null` — специальное значение "ничего"
- `undefined` — значение не определено

**Основные операторы:**
- `=` — присвоить значение
- `+` — сложить числа или соединить строки
- `++` — увеличить на 1
- `>` — больше

## **Практика с DOM:**

```javascript
// Создаем счетчик кликов
let clickCount = 0;
const counterElement = document.createElement('div');
const button = document.createElement('button');

button.textContent = 'Кликни меня!';
counterElement.textContent = `Кликов: ${clickCount}`;

button.addEventListener('click', function() {
    clickCount++;
    counterElement.textContent = `Кликов: ${clickCount}`;
    
    // Меняем цвет в зависимости от количества кликов
    if (clickCount > 10) {
        counterElement.style.color = 'red';
    } else if (clickCount > 5) {
        counterElement.style.color = 'orange';
    }
});

document.body.append(button, counterElement);
```

**Мини-проект:** Создать интерактивный счетчик с визуальной обратной связью.
